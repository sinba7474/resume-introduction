INTRODUCTION

具備 Adobe Photoshop、Illustrator、InDesign 等電腦平面設計工具使用經驗，過去曾參與多項行政企劃與專案管理工作。於宜蘭大學有機發展中心服務期間，負責資訊平台維運與專案執行，也參與地方創生及環境教育推廣，同時取得行政院環保署環境教育人員認證。其後轉任觀光工廠團體業務，負責客源開發、報價設計、行程安排與顧客關係維護，培養了跨部門協作及商務應用思維。

在職同時完成行銷與流通管理碩士學位，建立起以資料與市場觀察為核心的決策流程：

辨識問題 → 蒐集資訊 → 行為洞察 → 擬定策略 → 成效回饋與優化。

近年來，我投入 AI 應用與自媒體經營，透過 AI 協作完成多項工具型專案，如前端業務報價系統與資料整合型網站。同時經營主題式 Instagram 帳號，實際操作內容規劃、流量測試與受眾分析。雖非工程背景，但能以需求方角度定義流程與邏輯，由 AI 進行技術落實，形成 人類策略設計 × AI 技術產能 的協作開發模式。

同時經營主題式 Instagram 帳號，實際操作內容規劃、版面節奏與受眾分析，將平台數據作為調整視覺策略與文字風格的依據，持續測試「設計 × 行為 × 工具」之間的關係。

WEB / DATA VISUALIZATION

好鳥網：野鳥監測資料視覺化平台

串接全球 eBird 資料庫與自動化地圖技術，將野鳥紀錄投射至互動式地圖。 透過 Google Maps 介面呈現觀測差異，協助初學者迅速理解「有哪些鳥出現過」， 降低進入門檻，同時滿足進階玩家對資料精細度的需求。

角色：系統企劃／資訊架構設計
功能：篩選介面設計、地圖視覺風格規劃、文字與圖例說明

WEB APP / TOOL DESIGN

旅宿業團體報價系統

針對實際櫃檯與團體業務操作情境設計的 Web App，將房型、會議室與專案價等資訊整合， 提供 OTA 日曆選取、即時調整 JSON 報價表、匯出 PDF 報價單等功能， 讓原本依賴 Excel 的流程轉為視覺化操作，降低新人訓練成本並提升回覆效率。

角色：產品企劃／介面線框圖／需求彙整
功能：動態報價邏輯設計、RWD 介面規劃、操作流程示意
